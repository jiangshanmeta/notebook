# 配置ETag

ETag是用来对资源进行标识的响应头，可以看做是对Expire的补充。

当浏览器发起条件GET请求时，既可以使用If-Modified-Since对更新时间询问是否过期，也可以使用If-None-Match对唯一标识进行询问。如果两个请求头都存在，需要更新时间和ETag都不变才能返回304。

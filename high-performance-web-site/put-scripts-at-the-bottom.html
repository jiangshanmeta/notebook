# 将脚本放在底部

将脚本放在底部的的原因是脚本加载执行过程中会阻塞其他其他资源的加载。

之所以会阻塞其他资源的加载，有两个原因:

  • 脚本可能有DOM、样式等操作，因此浏览器需等待脚本执行确保页面功能正常
  • 多个脚本之间可能会有依赖关系，需要按顺序完成加载

# MVC框架框架安全

在框架中实施安全方案，比由程序员在业务中修复一个个具体的bug有更多的优势。首先，在框架中统一解决，可以大大节省人力成本。其次，避免在业务层修补带来的遗漏问题。最后，提供统一的安全方案，避免业务层实现不一致，也方便以后升级。
